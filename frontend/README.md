# MergenLite Frontend

React + Material UI frontend for SOW Generator.

## Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend will run on http://localhost:3000

## Backend

Make sure FastAPI backend is running on http://localhost:8000

## Features

- RFQ PDF upload
- Seating Chart PDF upload (optional)
- Pipeline selection (v2, v3, Hotel SOW Template)
- Real-time logs
- Generated PDF list
- Material UI design
