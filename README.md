# MergenLite v2.0

**Modern Government Contract Intelligence Platform**

Production-ready RFQ/SOW analysis powered by FastAPI + React + PostgreSQL (pgvector).

## Quick Start

```bash
docker compose up --build
```

- Frontend: http://localhost:3000
- Backend: http://localhost:8000/docs  
- Login: admin / mergenlite
